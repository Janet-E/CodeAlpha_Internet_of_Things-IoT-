Internet of Things (IoT) Embedded Systems Portfolio
Two software-based IoT implementations bridging automated marine telemetry analysis and system access control logic using pure Python.

Project Structure
-Offshore Marine Sensor Based Simulation: Real-time offshore environmental telemetry loop
-Smart Door Lock System: Host-based gateway access controller with acoustic alarms
-Smart Door Lock diagram: Matplotlib script to generate circuit schematics
-Marine_sensor_log.txt: Automated environmental history database

Script Overview & Execution

Task 2: Marine Environmental Monitoring Sensor:
Simulates real-time offshore sensor arrays tracking water temperature, dissolved oxygen levels, salinity concentrations, and pH scales. Includes automated threshold validation engines that flag environmental anomalies and stream continuous data points into a localized tracking file.

Task 3: IoT Smart Door Lock System Prototype:
Implements a host-based gateway access controller using computer sound peripherals to emulate physical alert buzzers. Monitors credential attempts against an exposure limit, logs access logs to disk, and triggers system lockdown states on unauthorized access thresholds.
